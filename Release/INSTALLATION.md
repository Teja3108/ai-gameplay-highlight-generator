# Installation Guide

1. Open `AI-Gameplay-Shorts-Generator-1.0.2-arm64.dmg`.
2. Drag **AI Gameplay Shorts Generator** to the **Applications** folder.
3. Eject the mounted disk image.
4. Open the app from Applications.

On first launch, macOS may show a security confirmation because this build is not yet notarized. Control-click the app, choose **Open**, then confirm.

The app starts its local backend and creator workspace automatically. It keeps projects and settings under your macOS Application Support folder. Closing the app stops its local services safely.
