# Quick Start

1. Double-click **AI Gameplay Shorts Generator**.
2. Open **Generate**, choose a gameplay recording, and select **Review before rendering**.
3. Wait for the local analysis to finish.
4. In **AI Review Studio**, use the timeline and source monitor to inspect each detected highlight.
5. Approve the clips you want, optionally refine their in/out points, then select **Render approved**.
6. Preview and save completed clips from **Review**. Your unfinished review decisions are saved automatically.

The application runs locally. Keep source recordings and generated clips on a disk with sufficient free space.
